pySTREAM
===============================

Overview
--------

Python implementation of the STREAM hydrological rainfall-runoff model

Installation / Usage
--------------------

To install use pip:

    $ pip install pystream


Or clone the repo:

    $ git clone https://github.com/martibosch/pystream.git
    $ python setup.py install
